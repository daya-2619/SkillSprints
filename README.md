# AI‑Powered Micro‑Learning Platform (skillsprint Clone)

This repository contains the full stack for a production‑ready mobile learning application built with:
- **Frontend**: React Native, Expo SDK, TypeScript, Expo Router, Zustand, React Query, Reanimated, etc.
- **Backend**: FastAPI, PostgreSQL, SQLAlchemy, Alembic, Redis, Celery, WebSockets, JWT auth.
- **Infrastructure**: Terraform for AWS, Docker, GitHub Actions CI/CD.
- **AI Services**: OpenAI / Gemini LLM integration, vector DB (Qdrant), RAG pipeline.

Refer to the `docs/` folder for architecture diagrams, API specs, UI wireframes, and the development roadmap.
