# UI Wireframes & Layouts - SkillSprint

*Note: Visual representations have been generated via the `generate_image` tool and are located in the `artifacts/` output.*

## 1. Onboarding Flow
- **Goal Selection**: Selectable tags using glassmorphism.
- **Experience Level**: Sliders/Toggles.
- **Progress**: A visual progress bar at the top with glowing indicators.

## 2. Main Feed (TikTok-style)
- **Video Surface**: Full screen, auto-playing.
- **Overlay**: Gradient overlay at the bottom for text readability.
- **Actions Bar**: Right-aligned vertical column for Like, Bookmark, Share, and "Ask Tutor".

## 3. Course Dashboard
- **Progress Ring**: SVG-based circular progress indicator.
- **Modules List**: Expandable accordion items for lessons and quizzes.

## 4. AI Tutor Chat
- **Bubbles**: Differentiated by color (User: Primary blue, AI: Surface dark grey).
- **Streaming Indicator**: Pulsing dots when waiting for SSE chunks.
