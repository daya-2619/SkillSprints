# Development Roadmap - SkillSprint

## Phase 1: Foundation (Weeks 1-4)
- Setup React Native repo, FastAPI repo, Terraform.
- Implement RBAC, JWT Auth, and DB schemas.
- Build basic UI layouts for login/signup and Home feed.

## Phase 2: Core Learning & Content (Weeks 5-8)
- Course creation dashboard for instructors.
- Video upload pipeline with S3/CloudFront.
- Implement TikTok-style feed component in React Native.
- Setup basic quizzes and flashcards UI.

## Phase 3: AI Integrations (Weeks 9-11)
- Setup pgvector and embedding ingestion pipeline via Celery.
- Build AI Tutor chat interface.
- Implement SSE backend routes for streaming LLM responses.

## Phase 4: Gamification & Polish (Weeks 12-14)
- Implement streaks, XP, and badges logic.
- Integrate push notifications.
- Optimize WebView for external labs.

## Phase 5: Launch & Analytics (Weeks 15-16)
- Setup Amplitude/Firebase analytics.
- Load testing and security audits.
- App Store / Google Play submissions.
